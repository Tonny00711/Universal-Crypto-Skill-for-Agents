## 🚀 快速开始

### 1. 注册工具
将 `schema.json` 中的内容复制到你的 Agent 配置或 OpenAI API 调用的 `tools` 参数中。

### 2. 部署逻辑
在你的本地环境或云函数中运行 `action.py`。

```python
from skills.crypto_price.action import get_crypto_price

# Agent 识别到意图后会自动调用：
result = get_crypto_price("BTC")
print(result) 
# Output: {"symbol": "BTCUSDT", "price": 68544.31, "source": "Binance", ...}
```
